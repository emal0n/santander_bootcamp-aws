# RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS

Data: 27/11/2025
Empresa: Abstergo Industries
Responsável: Edmundo Neto

## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries, realizado por Edmundo Neto. 
O objetivo do projeto foi selecionar e implantar 3 serviços AWS com foco em redução imediata de custos, garantindo ao mesmo tempo estabilidade, segurança e escalabilidade para as operações.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com objetivos específicos.

### Etapa 1:
- Amazon S3 (Simple Storage Service)
- Foco da ferramenta: Armazenamento escalável e de baixo custo
- Descrição de caso de uso:
  Migração de arquivos estáticos e backups da infraestrutura local para o Amazon S3, utilizando classes de armazenamento como S3 Standard-IA e S3 Glacier para reduzir custos e garantir alta durabilidade dos dados.

### Etapa 2:
- AWS Lambda
- Foco da ferramenta: Execução de código sob demanda sem servidor (serverless)
- Descrição de caso de uso:
  Automação de rotinas internas, como processamento de logs e execução de scripts de manutenção, eliminando a necessidade de servidores dedicados e reduzindo custos operacionais.

### Etapa 3:
- Amazon EC2 com instâncias Spot
- Foco da ferramenta: Capacidade computacional de baixo custo
- Descrição de caso de uso:
  Migração de workloads não críticos para instâncias Spot, reduzindo drasticamente custos de processamento para tarefas como renderização, análise de dados e workloads temporários.

## Conclusão
A implementação das ferramentas na empresa Abstergo Industries proporcionou redução de custos imediata, aumento da eficiência operacional e maior flexibilidade tecnológica.
Recomenda-se a continuidade do uso das soluções adotadas e a avaliação constante de novos serviços AWS que possam otimizar ainda mais a infraestrutura da empresa.
